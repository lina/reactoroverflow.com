## Requirements

- Node 0.12.x
- NPM 2.11+
- Angular
- elasticsearch
- Express 4.x

## Development

### Installing Dependencies

Setting up ElasticSearch

```sh
brew update && brew install elasticsearch
```

From within the root directory:

```sh
npm install -g bower
npm install
bower install
node index.js
```
